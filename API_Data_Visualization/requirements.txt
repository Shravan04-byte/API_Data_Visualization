# 🌦️ Weather Forecast Visualization - Internship Task 1

## 📌 Task Objective
Use Python to fetch data from the OpenWeatherMap API and visualize it using Matplotlib/Seaborn.

## 🔧 Tech Stack
- Python
- Requests (for API calls)
- Matplotlib and Seaborn (for data visualization)

## 📊 Features
- Fetch 5-day forecast data using OpenWeatherMap
- Plot a temperature vs. time line chart
- Save visualization as `graph_output.png`

## 📝 How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the script:
python weather_visualization.py

3. Output:
- Weather data is saved to `sample_output.json`
- A graph is saved as `graph_output.png`
